# PHP-Sword

> 基于EasySwoole的PHP常驻内存开发框架

## 主要特性

* 采用`PHP7`强类型（严格模式）
* 支持更多的`PSR`规范
* 基于Swoole常驻内存
* 协程TCP、UDP、WEB_SOCKET 服务端

## 安装

~~~
composer require php-sword/storage
~~~
